# SIXPoetryFate
一款小巧的诗歌赏析APP

<h3>功能描述：</h3>
<div>
  本地提供500首诗歌，名家生平<br/>
  提供收藏、编写感想<br/>
  提供网络搜索，下载本地<br/>
  古代歌曲赏析，诗歌论坛浏览<br/>
  可自定义APP风格
</div>
