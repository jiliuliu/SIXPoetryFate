# SIXPoetryFate
一款小巧的诗歌赏析APP

<h3>功能描述：<h3>
<p>  本地提供500首诗歌，名家生平</p>
<p>  提供收藏、编写感想</p>
<p>  提供网络搜索，下载本地</p>
<p>  古代歌曲赏析，诗歌论坛浏览</p>
<p>  可自定义APP风格</p>
